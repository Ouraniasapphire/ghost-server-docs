## Log into the domain


### To navigate to the domain login screen 
- See [This article](https://www.petenetlive.com/kb/article/0001764) or [Registering to the domain](../registerDomain/README.md)

### To login 
1. Enter domain name: "kctc.ghost.local" & continue
2. Enter username: See [Users](../users/README.md) & continue
3. Enter fancy password (If password has been changed, use that)
4. Restart computer

### After restart:
1. Enter username: see [Users](../users/README.md)
2. Enter password

### If password is asking to be changed
- For password requirements, see [Users](../users/README.md)
